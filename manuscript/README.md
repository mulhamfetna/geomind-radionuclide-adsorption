# Manuscript — withheld pending publication

The manuscript reporting these results (`manuscript-submission.md/.docx/.pdf`, the section drafts,
the outline and Table 3) is **deliberately not included in this repository**. It is withheld
pending journal submission and peer review, and will be added in a later release once published.

Other documents in this repository may link to those files; those links are intentionally
unresolved for now.

**What *is* here:**

| File | Purpose |
|---|---|
| `figures.py` | Generates every figure directly from the live analysis pipeline |
| `figures/*.png` | The rendered figures (also used by the Virtual Lab) |
| `figures/figures.pdf` | All figures in one document |

**Nothing needed to reproduce the results is missing.** The data (`data/`, `knowledge_base/`), the
analysis code (`src/geomind/`), the audited pools, the forward model and the Virtual Lab
(`app/`, `notebook_lab/`) are all present — every number in the paper can be regenerated from
them. See the [main README](../README.md) and
[`reports/plan-vs-achieved.md`](../reports/plan-vs-achieved.md).
