## Priority if you only fetch a few

1. **#2 Next Sustainability** — the only activator-molarity (1–5 M) axis available anywhere, with
   Cs *and* Sr, plus the NMR we now know is the key descriptor.
2. **#1 Baek** — clean Langmuir + Freundlich *with* CEC and Si/Al.
3. **#5 Functionalized geopolymer foams** — most-cited, and foams are a distinct morphology class.

Everything else is a bonus.
